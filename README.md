# demo-finos-ca

Demo agents for the Kagenti platform, showcasing Google ADK multi-agent orchestration with Skills.

## Agents

### [Story Skills Agent](a2a/story_skills_agent/)

A multi-agent story writing pipeline built with Google ADK, demonstrating:
- Multi-agent orchestration (Sequential, Parallel, Loop agents)
- ADK Skills (genre-guide, story-structure, character-builder)
- LlamaStack integration via LiteLlm
- A2A protocol for Kagenti discovery
